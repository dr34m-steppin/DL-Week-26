# Testbench Checklist

Use this checklist during grading/demo.

- [ ] App launches with `streamlit run frontend/streamlit_app.py`
- [ ] Dataset loads successfully from Kaggle
- [ ] Single PDF upload works
- [ ] Chat response includes citations from uploaded content
- [ ] Weak-skill diagnosis appears for selected student
- [ ] Targeted micro-quiz is generated
- [ ] Auto-grading produces score + feedback
- [ ] Mastery/SnapScore updates after quiz result
- [ ] Professor can validate skill map
- [ ] Professor can override risk flag
- [ ] Professor can confirm/override grade
- [ ] Professor decisions persist in `data/professor_overrides.json`
